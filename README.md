# Aoidh Hugo Theme

Aoidh is a simple theme for [Hugo](https://gohugo.io/).
It's mostly just CSS.

_Aoidh_ ([pronounced _eye_](http://www.clanmackay.ca/history.html)) is Gaelic for _fire_.
It is anglicized as Hugh, which is also the English version of the German name [Hugo](https://gohugo.io/).
